# opensprite

it's a tool to animate a single sprite into a full sheet using ai. 

### how to run
1. open your terminal in this folder
2. run `./serve.sh`
3. go to `http://localhost:8080` in your browser

### how to use
- paste your openrouter api key in the sidebar
- upload your character image
- type the action you want (like "walking" or "jumping")
- click generate and wait for the frames to appear
- download as a spritesheet or zip when you're done

### what it is
- **no backend:** everything runs in your browser, stays private
- **ai powered:** uses gemini 1.5 flash via openrouter
- **flexible:** auto-split or manual cropping included
- **simple:** no npm or complex setup needed

### why it exists
hand-drawing animation frames takes forever. this uses ai to handle the heavy lifting while keeping you in control.
